# moviesearch-backend

#### 1. Clone the repo

```
git clone
```

#### 2. Install dependencies

```
npm install
```

#### 3. Rename .env-sample file to .env and add desired PORT variable value in .env file

```
PORT=8080
```

#### 4. Run development server

```
npm run dev
```

#### 5. Run Production server

```
npm run start
```
